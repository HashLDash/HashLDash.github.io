Implementando a física do jogo

Nesse vídeo nós vamos aprimorar a tela de jogo, deixando o player com um tamanho definido e posicionado para o início do jogo. Vamos também implementar a física do jogo, adicionando o efeito da gravidade no jogador e fazendo com que ele pule ao clicar (ou tocar) na tela.

Clique na imagem para assistir a aula no YouTube.

Chegou o momento de suas vidas em que vocês vão utilizar os conhecimentos de Física!

Antes disso, vamos arrumar a nossa tela de jogo. Vamos editar o arquivo hashbird.kv e adicionar um FloatLayout na tela Game, e adicionar o Player nesse Layout.

Depois precisamos configurar o Player, para que ele tenha um tamanho e posicionamento adequados. 

#hashbird.kv

<Game>:

    FloatLayout:

        Player:

            id:player

            allow_stretch:True

            size_hint:None,None

            size:!largura!,!altura!

            x:!posicaoX!

            y:!posicaoY!

Primeiro, colocamos o id player, para que possamos acessar esse Widget pelo Python. O 'allow_stretch' faz com que a imagem tome o tamanho do Widget, esticando a imagem se necessário. Depois colocamos o tamanho e posição que quisermos, indicados por !largura!,!altura!,!posicaoX! e !posicaoY!. Experimente alguns valores e coloque os que achar mais interessante para o seu jogo!

Vamos agora para o arquivo main.py, onde vamos implementar a física do jogo.

Este é um jogo muito simples, fisicamente falando, pois só teremos que implementar a gravidade e colisões. Nesta aula vamos começar colocando a gravidade.

Vamos criar um método que vai ser responsável por atualizar a posição do Player, chamado update. E vamos agendar a execução dele com o Clock periodicamente. Esse agendamento deve ser feito quando entrarmos na tela de jogo. 

#main.py

from kivy.clock import Clock



class Game(Screen):

    def on_enter(self,*args):

        Clock.schedule_interval(self.update,1/30)

Neste caso, agendamos a execução do self.update 30 vezes por segundo. Depois vamos adicionar uma propriedade 'speed' que vai ser a velocidade do Player. 

#main.py

from kivy.properties import NumericProperty



class Player(Image):

    speed = NumericProperty(0)

E, finalmente, a parte da Física. Lembra que a taxa de variação da velocidade no tempo é a aceleração? No nosso joguinho, o Player está sob ação da gravidade somente. Então a velocidade do Player só vai variar conforme a gravidade do jogo. Então, basta atualizarmos a velocidade do Player e calcularmos a posição dele. 

#main.py

    def update(self,*args):

        self.ids.player.speed += -gravidade * 1/30

        self.ids.player.y += self.ids.player.speed * 1/30

Lembre-se de atribuir um valor que ache adequado para a gravidade.

Agora vamos fazer o Player pular. Para isso, só precisamos estabelecer uma velocidade para o Player quando tocarmos, ou clicarmos, na tela. 

#main.py

    def on_touch_down(self,*args):

        self.ids.player.speed = velocidade

Lembrando que você deve atribuir um valor para 'velocidade'.

Pronto!! Agora é só sair pulando com a bolinha!
