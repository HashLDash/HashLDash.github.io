Como utilizar Markdown

Este é uma referência aos recursos do Markdown. Markdown é uma linguagem simples utilizada para formatar textos. Diversos sites e aplicativos (como o HashForum) utilizam esta linguagem para formatar posts, criar sites e etc.

Confira abaixo as aulas do curso e uma breve descrição do conteúdo abordado em cada aula.
