Sintaxe do Markdown

Nesta aula você vai aprender a criar textos formatados utilizando a linguagem Markdown e sua sintaxe.

Clique na imagem para assistir a aula no YouTube.

Para criar um texto formatado em Markdown, utilize a seguinte sintaxe para criar cada elemento:

Criando títulos de seções

Para criar um título, basta colocar o símbolo '#' em frente ao texto do título, da seguinte maneira: 

# Novo título

Para criar subtítulos, basta repetir o símbolo na quantidade do subtítulo desejada: 

## Novo subtítulo



### Novo sub subtítulo

Criando um bloco de código

Para formatar um código, utilize a seguinte sintaxe: 

```nomeDaLinguagem

meu código revolucionário

```

Se você indicar o nome da linguagem do código, alguns formatadores irão colorir seu código conforme a linguagem (syntax highlighting). Exemplo, para um código em Python, utilize: 

```python

def test():

    print('Python is awesome!')

```
