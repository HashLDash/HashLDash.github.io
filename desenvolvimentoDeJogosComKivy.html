Desenvolvimento de jogos com Kivy

Este curso fala sobre o desenvolvimento de jogos utilizando o framework Kivy. Para acompanhar este curso é necessário um conhecimento básico de Python e Kivy, que pode ser adquirido assistindo os cursos: Curso básico de Python e Interface gráfica com Kivy. Vamos desenvolver um jogo 2D baseado no Flappy Bird como projeto.

Confira abaixo as aulas do curso e uma breve descrição do conteúdo abordado em cada aula.

Como criar um jogo

Nesse vídeo eu faço uma introdução sobre esta nova etapa do cuso de Python na prático, onde o enfoque é a criação de um jogo. Para desenvolver esse jogo, vamos utilizar a linguagem Python e a framework Kivy para a interface gráfica. Este vídeo trata da parte inicial que é a criação da tela e a inserção do player no jogo.

Criando o menu do jogo

Nesse vídeo nós vamos criar telas para o nosso jogo, sendo uma tela de Menu e outra a tela do Jogo. Vamos construir uma tela de menu com uma imagem de fundo e botões customizados. Depois vamos criar a funcionalidade dos botões para sair do jogo e para iniciar o jogo. O botão de iniciar irá trocar a tela para a tela de jogo.

Implementando a física do jogo

Nesse vídeo nós vamos aprimorar a tela de jogo, deixando o player com um tamanho definido e posicionado para o início do jogo. Vamos também implementar a física do jogo, adicionando o efeito da gravidade no jogador e fazendo com que ele pule ao clicar (ou tocar) na tela.

Tela de game over

Nesse vídeo nós vamos criar a tela de game over do jogo e criar as condições de derrota. A tela de game over terá dois botões para tentar novamente e para sair do jogo. Vamos resetar as condições do jogo para o estágio inicial, para que a nova rodada seja iniciada.

Criando os obstáculos no jogo

Nesse vídeo nós vamos criar os obstáculos do jogo e fazer com que eles se movam na direção do player. Vamos também reiniciar os obstáculos para que o jogo possa iniciar sem que os obstáculos do jogo passado apareçam na tela.

Obstáculos aleatórios e placar do jogo

Nesse vídeo nós vamos deixar o jogo um pouco mais interessante, criando um obstáculo que tem um buraco no qual o player deve passar. Vamos também implementar a aleatoriedade para que o buraco apareça em lugares diferentes. Além disso, vamos incluir um placar no jogo para mostrar os pontos.

Algoritmo AABB para detecção de colisões

Nesse vídeo nós vamos implementar a detecção de colisões no nosso jogo. Vamos aprender como é feita a detecção de colisões e suas particularidades. Depois vamos implementar o algoritmo AABB (Axis-aligned bounding box) no jogo.

Lógica de pontuação do jogo

Nesse vídeo nós vamos implementar a lógica de pontuação do jogo. A cada obstáculo ultrapassado, o jogador ganhará um ponto. Com isso vamos relembrar sobre as propriedades do Kivy e aprender um pouco mais sobre como o Kivy funciona pra poder solucionar alguns problemas comuns dos aplicativos Kivy.

Preparando seu aplicativo para a Play Store!

Nesse vídeo eu demonstro os passos para compilar um APK que pode ser colocado na Google Play Store! Vamos utilizar o buildozer para gerar o APK, criar chaves criptográficas e assinar o APK em conformidade com as regras da Google Play Store.
