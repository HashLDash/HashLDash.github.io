Kivy - Sintetização de voz no android com Plyer

Nesse vídeo vamos aprender sobre o módulo Plyer, criado pelos desenvolvedores do Kivy. Com ele é possível acessar as funções nativas do Android e de outras plataformas. Vamos utilizar ele para acessar o motor de sintetização de voz do Android para que o nosso chatbot fale. Vamos aprender também a incluir pacotes novos no projeto do android.

Clique na imagem para assistir a aula no YouTube.

Para utilizar a sintetização de voz do android, precisamos acessar a API nativa que está em Java. Para isso os desenvolvedores do Kivy criaram um wrapper para tornar a API Java acessível pelo Python. Além disso, criaram um módulo chamado Plyer, que utiliza esse wrapper, para tornar certas funcionalidades do Android mais facilmente acessíveis para os desenvolvedores.

Para utilizar este módulo no android, precisamos adicioná-lo no buildozer.spec como um requisito do aplicativo. 

# buildozer.spec

requirements = kivy, plyer, jnius, android

No meu caso, precisei adicionar o módulo 'jnius' e 'android' manualmente, pois o 'plyer' depende deles para funcionar corretamente.

Após esta inclusão, podemos acessar as funcionalidades do módulo, como o tts (Text-to-Speech). 

# main.py

from plyer import tts



tts.speak('Frase a ser dita')

Por enquanto, só a língua inglesa está disponível pelo Plyer. Mas o suporte para outras línguas está bem próximo. A sintaxe prevista para alterar a língua da voz é: 

tts.speak('Frase a ser dita', language=lang)

No vídeo eu mostro como fazer essas alterações e implementá-las no assistente virtual. Eu também compilei e testei no Android, confira!
