Kivy - Criando um Popup

Nessa aula vamos aprender a criar um Popup no Kivy. Vamos implementar um Popup de confirmação de saída para o nosso aplicativo.

Clique na imagem para assistir a aula no YouTube.

Para criar um Popup no kivy, basta utilizarmos o Widget 'Popup', disponível na biblioteca padrão. Para importar, basta executar: 

from kivy.uix.popup import Popup

O Popup é basicamente composto de um título, passado pelo argumento 'title' e seu conteúdo, passado pelo argumento 'content'. O títle é uma string que será exibida no topo da janela do nosso Popup. Já o conteúdo, é um widget ou layout que será apresentado no corpo da janela.

Para criar um Popup, basta executar: 

pop = Popup(title="Título",content=layout)

Para que o Popup seja exibido, é preciso executar: 

pop.open()

Para fechar o Popup: 

pop.dismiss()
